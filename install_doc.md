# 安装文档

## 环境需要

1. 能打开pdf的软件
2. 能打开doc,xls,ppt之类的软件（如果没有，则暂时不能打印doc，xls等），也推荐LibreOffice
3. node 18+
4. python 3.7 +
5. windows
6. 已经装好的打印机驱动
7. 确保 python，node已经在环境目录内
8. 8080端口，3000,5050端口没有被占用


## 安装

1. 解压项目文件，最好不要有中文目录
2. 运行install_server.cmd
  
  
## 运行 

1. 运行run_server.cmd
2. 服务器已经在8080端口运行。
